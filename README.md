# 🧠 PulseIQ — Intelligent Health Assessment System

PulseIQ is a multi-stage health assessment platform that analyzes user vitals and symptoms to generate real-time, domain-specific health insights across cardiovascular, neurological, respiratory, mental, and environmental factors.

---

## ⚡ Features

- 🕒 **Time-Based Monitoring**  
  Capture health parameters at 0, 5, 10, and 15 minutes  

- ❤️ **Vitals Analysis**  
  Tracks Heart Rate, Blood Pressure, SpO2, and Temperature  

- 🧩 **Multi-System Evaluation**
  - Cardiovascular  
  - Neurological  
  - Respiratory  
  - Mental Health  
  - Weather Impact  

- 📊 **Smart Risk Assessment**
  - Primary State  
  - Current State  
  - Risk Tendencies  
  - Symptom-based scoring (%)  

- 🧾 **Symptom Checker**  
  Dynamic symptom selection with intelligent condition mapping  

---

## 🖥️ Tech Stack

- **Frontend:** HTML, CSS (Neon UI), JavaScript  
- **Backend:** Python (Flask/Django)  
- **Core Logic:** Rule-based health analysis engine  

---

## 🚀 How It Works

1. User enters initial health parameters  
2. System records time-based vitals (0–15 minutes)  
3. User selects symptoms  
4. Engine processes:
   - Vitals interpretation  
   - Symptom correlation  
   - Risk scoring  
5. Results are generated across multiple health domains
